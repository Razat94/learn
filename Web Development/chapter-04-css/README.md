# Chapter 4: Applying Styles to Web Content

## Overview
This chapter introduces CSS, the language used to style and format HTML content.

Students learn how to apply styles directly to HTML, create external stylesheets, and use fonts, colors, spacing, and basic selectors to control the look of a web page.

## Objectives
By the end of this chapter, students will be able to:

- Explain the purpose of CSS
- Apply inline styles to HTML elements
- Create and link an external stylesheet
- Use basic CSS selectors
- Apply colors, font styles, and spacing
- Use web fonts to improve page design

## Key Concepts

### HTML vs CSS
HTML provides structure. CSS controls presentation.

### Inline Styles
Styles can be written directly on an HTML element using the `style` attribute.

### External Stylesheets
CSS is usually stored in a separate `.css` file and linked to HTML using the `<link>` element.

### Selectors
Selectors tell the browser which HTML elements to style.

### Web Fonts
Web fonts allow developers to use fonts that may not already exist on the user’s device.

## One-Line Summary
CSS is used to control how HTML content looks on the page.
