# Chapter 9: Submitting Data to a Web Server

## 📌 Overview
This chapter explains how form data is sent from the browser to a web server for processing.

Students learn how forms communicate with servers using HTTP methods and how data is structured and transmitted.

---

## 🎯 Objectives
- Understand how form submission works
- Learn GET vs POST methods
- Use form attributes correctly
- See how data is sent to a server

---

## 💡 One-Line Summary
Forms send user data to a server using HTTP methods like GET and POST for processing.
