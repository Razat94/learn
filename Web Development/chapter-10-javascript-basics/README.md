# Chapter 10: JavaScript Basics

## 📌 Overview
This chapter introduces JavaScript as a programming language used to make web pages interactive and dynamic.

Students learn how to perform operations, write simple programs, and modify web page content using the Document Object Model (DOM).

---

## 🎯 Objectives
- Add JavaScript to make web pages interactive
- Perform operations on numbers and strings
- Use loops for repetitive tasks
- Modify elements using the DOM

---

## 💡 One-Line Summary
JavaScript allows developers to add logic, interactivity, and dynamic behavior to web pages.
