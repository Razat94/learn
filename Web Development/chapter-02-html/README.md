# Chapter 2: Creating Web Content in HTML

## 📌 Overview
Students will learn how to create real web pages using HTML and connect them together into a simple website.

## 🎯 Learning Objectives
By the end of this chapter, students will:

- Create a valid HTML document
- Use headings, paragraphs, and lists
- Add navigation to link pages together
- Structure content using semantic HTML

## 🧠 Key Concepts

### 1. HTML Structure
- <!DOCTYPE html>
- <html>, <head>, <body>

### 2. Content Elements
- Headings (h1–h6)
- Paragraphs
- Lists
- Links

### 3. Navigation
- Anchor tags (<a>)
- Relative paths

### 4. Semantic HTML
- <header>, <nav>, <main>, <section>, <footer>

## 🚀 Outcome
Students build a small multi-page website.
